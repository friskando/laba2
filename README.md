# Задание 1

### Текст задачи

На основе данного списка строк получить новый, где к каждой строке в начале

дописан заданный символ. (list.map)

### Алгоритм решения

1. Выбираем задание из 2 предложенных
2. Запрашиваем у пользователя  количество строк через отдельную рекусривную функции с проверкой на ввод (реализована проверка через словарь и кейс попытку преобразовать в int)
3. после успешной проверки запрашиваем значение К (нам без разницы какое оно будет хоть string хоть int) оно всегда strig
4. выводим сколько пользователь заказал строк
5. добавляем в список значения  помощью for (так же значения будут string) 
6. черtз функцию uni (униикация\*) используем list map для извлечения значений из списка и преобразовываем в другую функцию add для соединения двух стринговых значений и возвращаем новый список значения с + К
7. &nbsp;выводим через через функцию вывода красиво

# Задание 2

### Текст задачи

Список содержит строки. Найти количество строк заданной длины (list.fold)

### Алгоритм решения

1. Выбираем задание из 2 предложенных
2. Аналогично с первым запрашиваем кол строк
3. выводим сколько пользователь попросил строк для ввода
4. аналогично первому заданию добавляем значения в список
5. после получения списка с знаениями пользовтаеля передаем этот список в функцию для подсчета длины его элементов
6. запрашиваем у пользователя с какой длиной строку будем искать аналогично первому заданию проверяем на корекктный ввод
7. с помощью вспомогательной функции len сравниваем сколько строк в списке = значению с искомым (через аккумуляторную анонимную функцию)
8. в спомогательной функции через list fold считаем сколько значений в списке имеют равную длину искомой
9. красиво выводим
